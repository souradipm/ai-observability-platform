# AI Observability Platform

A production-grade AI platform for observability, incident investigation, and autonomous operations built with Java, Spring Boot, Agentic AI, RAG, Kafka, OpenTelemetry, and Kubernetes.

## 🚀 Vision

This project demonstrates how to build an enterprise-grade AI platform from scratch using modern software architecture and production engineering practices.

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Maven
- Lombok
- Spring Actuator

## 📌 Current Status

### Milestone 1 - Platform Bootstrap ✅

- Modular package structure
- Health endpoint
- Generic API response
- Architecture Decision Records (ADR)
- GitHub project initialized
